# /talk - SenseVoice 음성 입력 실행

SenseVoice 기반 음성-텍스트 변환을 **백그라운드**에서 실행합니다. CMD 창 없이 조용히 동작하며, 비프음으로 상태를 알려줍니다.

## 트리거

- `/talk` - 음성 입력 서비스 시작
- 키워드: SenseVoice, 음성 입력, 빠른 STT, 한국어 음성인식

## 실행 절차

1. 이미 실행 중인지 확인 (pythonw.exe sensevoice_bg.pyw)
2. 실행 중이 아니면 백그라운드에서 시작
3. 비프음 2회 = 준비 완료

## 명령어

```bash
# 백그라운드 실행 (창 숨김)
wscript "C:\Users\PC\tools\SenseVoice\start_hidden.vbs"
```

## 사용법

**어디서든** (VS Code, 메모장, 브라우저 등):

| 동작 | 방법 | 소리 |
|------|------|------|
| 녹음 시작 | **백틱(`)** 키 | 삐 (높음) |
| 녹음 중지 | **백틱(`)** 키 | 삐 (낮음) |
| 변환 완료 | 자동 | 삐삐 (2회) |
| 오류 | - | 삐 (낮고 길게) |

**결과는 커서 위치에 자동 입력됩니다!**

## 비프음 가이드

| 소리 | 의미 |
|------|------|
| 삐삐 (높음 2회) | 모델 로드 완료, 준비됨 |
| 삐 (800Hz) | 녹음 시작 |
| 삐 (600Hz) | 녹음 중지, 변환 시작 |
| 삐삐 (1000+1200Hz) | 변환 성공, 텍스트 입력됨 |
| 삐 (300Hz 길게) | 오류 발생 |

## 종료 방법

작업 관리자에서 `pythonw.exe` 종료 또는:
```bash
taskkill /F /IM pythonw.exe
```

## 특징

- **백그라운드 실행**: CMD 창 없음
- **속도**: 70ms/10초 오디오 (Whisper 대비 15배)
- **자동 입력**: 커서 위치에 바로 입력
- **언어**: 한국어 고정
